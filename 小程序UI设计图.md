# 🎨 社区食安AI小卫士 - 小程序UI设计图

> AI赋能的社区食品安全共治平台 - 界面设计规范

## 📱 设计理念

### 核心设计原则
- **简洁易用**: 界面简洁明了，操作流程清晰
- **安全可信**: 使用蓝绿色调体现安全感和可信度
- **智能友好**: AI功能突出但不突兀，自然融入用户体验
- **社区温暖**: 体现社区共治的温暖氛围
- **适老化设计**: 考虑不同年龄段用户的使用习惯

### 色彩方案
```
主色调: #2E8B57 (海绿色) - 代表安全、健康
辅助色: #4169E1 (皇家蓝) - 代表科技、智能
强调色: #FF6B35 (橙红色) - 代表警示、重要
背景色: #F8F9FA (浅灰白) - 清洁、简约
文字色: #2C3E50 (深蓝灰) - 易读性
```

### 字体规范
```
标题字体: 18-20px, 加粗
正文字体: 14-16px, 常规
辅助文字: 12-14px, 浅色
按钮文字: 16px, 加粗
```
